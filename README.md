# Azra'ya — Kasımpatı Temalı Küçük Site

Bu proje tek sayfalık, mobil uyumlu bir sayfadır. Arka plan kasımpatı buketi görseli içerir; başlık olarak **Azra'ma** gösterilir ve sağ/alt tarafta bir not kağıdı içinde kısa şiir yer alır.

## Kullanım
1. Bu üç dosyayı (index.html, styles.css, README.md) aynı dizine koy.
2. `index.html` dosyasını doğrudan tarayıcıda açarak test edebilirsin.
3. Arka plan görselini değiştirmek istersen, `styles.css` içinde `--bg-url` değerini kendi resim URL'inle değiştir:
   ```css
   --bg-url: url('path/veya-yeni-gorsel.jpg');
